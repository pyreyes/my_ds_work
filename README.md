### [Restaurant Reviews Analysis](https://github.com/pyreyes/restaurant_reviews)
Collaborated to create a restaurant recommendation system using user reviews on Zomato. Developed an LDA based topic model for extracting the main themes found on text user reviews and performed hyper-parameter tuning to extract topics with highest coherence score.
